A widget to display the twitter remote widget from http://twittercounter.com/

== Description ==

A widget to display the twitter remote widget. Options supported:
Title, 2 colors, and username.
Note: this plugin requires curl and access to api.twittercounter.com (for fetching ID's from twitter).

== Installation ==

1. Upload the files to you plugin directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Choose a sidebar for your widget
1. Set your username, and preferred colors
1. Activate
